## Installation

To install the 3dvr-agent, follow these steps:

1. Clone the repository:
   bash
   git clone https://github.com/3dvr-agent/3dvr-agent.git
   

2. Navigate to the project directory:
   bash
   cd 3dvr-agent
   

3. Install the dependencies:
   bash
   npm install
   

4. Run the agent:
   bash
   npm start
